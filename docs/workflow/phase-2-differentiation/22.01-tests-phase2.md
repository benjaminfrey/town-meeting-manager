# Session 22.01 — Tests: All Phase 2 Features

**Phase:** 2 — Differentiation | **Block:** 22-23 — Testing & Deployment
**Dependencies:** 16.02, 17.02, 18.01, 19.01, 20.01, 21.02
**Estimated tasks:** 12

---

## Description

Write comprehensive tests for all Phase 2 features: AI minutes generation, audio recording and transcription, warrant articles, Twilio SMS with TCPA compliance, resident accounts and straw polls, and React Native mobile app functionality. This session covers integration tests, component tests, and end-to-end tests to verify all Phase 2 features work correctly individually and together.

## Tasks

1. Write integration tests for Claude API minutes generation: correct style output, section formatting, streaming
2. Write component tests for AI minutes review: accept/reject/edit sections, regeneration, accuracy checklist
3. Write integration tests for audio recording: segment creation, upload, playback
4. Write integration tests for transcription pipeline: audio -> transcript -> AI minutes integration
5. Write component tests for warrant article management: create, number, generate PDF
6. Write integration tests for Twilio SMS: TCPA consent, opt-out, quiet hours, delivery tracking
7. Write component tests for resident registration and straw poll voting
8. Write E2E test for straw poll lifecycle: create poll -> notify residents -> vote -> view results
9. Write integration tests for React Native app: PowerSync sync, voting, audio recording
10. Write E2E test for AI minutes: complete meeting -> generate AI draft -> review -> approve
11. Write E2E test for SMS notification: resident subscribes -> opt-in -> receives SMS -> opts out
12. Run full test suite (Phase 1 + Phase 2) and verify all tests pass
