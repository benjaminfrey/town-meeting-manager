# Session 04.02 — Board Management

**Phase:** 1 — MVP | **Block:** 04 — Town Profile & Board Management
**Dependencies:** 04.01
**Estimated tasks:** 9

---

## Description

Build the board management interface where admins create, edit, archive, and configure boards and committees. Each board can override town-level meeting defaults (formality, minutes style) and configure quorum rules. Boards are never deleted -- only archived -- to preserve historical meeting and minutes records.

## Tasks

1. Create BoardListPage at /boards route showing all boards for the town (DataTable with name, type, member count, status)
2. Create BoardDetailPage at /boards/:boardId with board info, member roster, and meeting history
3. Create AddBoardDialog for creating new boards (name, type, member count, elected/appointed, formality override, minutes style override)
4. Create EditBoardDialog for modifying board settings
5. Create ArchiveBoardDialog with confirmation (boards cannot be deleted, only archived -- meetings/minutes are preserved)
6. Implement board-level meeting formality and minutes style overrides (fall back to town defaults if not set)
7. Implement quorum configuration per board (default: simple majority >50% of seated members, overridable)
8. Create board settings panel for configurable motion display format per advisory 2.2 Q13 decision
9. Verify: boards list, create, edit, archive work correctly, overrides fall back to town defaults

TASK 7: Implement quorum configuration
- Each board has quorum settings:

- Default is simple majority: >50% of seated members (member_count), rounded up

- Create a QuorumConfigEditor component in the EditBoardDialog:

  - Number input for fixed quorum (shown only when 'fixed_number' selected, max = member_count)

- Create a utility function calculateQuorum(memberCount, quorumType, quorumValue) in packages/shared

  - two_thirds: Math.ceil(memberCount * 2 / 3)
  - three_quarters: Math.ceil(memberCount * 3 / 4)
  - fixed_number: quorumValue (validate <= memberCount)
- Display the quorum on BoardDetailPage: "Quorum: 3 of 5 (simple majority)"
