# Session 01.01 — Monorepo Initialization

**Phase:** 1 — MVP | **Block:** 01 — Infrastructure & Monorepo Foundation
**Dependencies:** None
**Estimated tasks:** 12

---

## Description

Initialize the pnpm monorepo with Turborepo build orchestration, creating the four-package workspace structure (web, mobile, api, shared) with strict TypeScript configuration and shared path aliases. This establishes the foundation that all subsequent sessions build upon.

## Tasks

1. Verify git repository exists (.git directory present)
2. Create root package.json with "town-meeting-manager" name, private: true, workspaces config
3. Create pnpm-workspace.yaml defining packages/* workspace
4. Create packages/ directory structure: web/, mobile/, api/, shared/
5. Create each package's package.json with proper naming (@town-meeting/web, @town-meeting/mobile, @town-meeting/api, @town-meeting/shared)
6. Create tsconfig.base.json with strict TypeScript settings, path aliases, composite project references
7. Create per-package tsconfig.json extending base config
8. Install and configure Turborepo (turbo.json) with build, dev, lint, test, typecheck pipelines
9. Create .gitignore (node_modules, dist, .turbo, .env*, coverage, etc.)
10. Create .nvmrc with Node.js 20 LTS
11. Create .npmrc with strict-peer-dependencies=true, auto-install-peers=true
12. Run pnpm install and verify workspace linking works

## Prompt

```
You are setting up a pnpm monorepo with Turborepo for the Town Meeting Manager project — a municipal meeting management platform for Maine towns. Read docs/advisory-resolutions/3.1-monorepo-structure.md for the confirmed package structure, then implement the following.

PROJECT CONTEXT:
- Monorepo with 4 packages: web (React + React Router v7), mobile (React Native/Expo, Phase 2), api (Fastify), shared (types, schemas, constants, utils)
- TypeScript everywhere — strict mode, composite project references
- pnpm workspaces for dependency management
- Turborepo for build orchestration and caching
- Node.js 20 LTS

TASK 1: Verify the git repository
- Confirm .git/ exists in the project root
- If it does not exist, run `git init`

TASK 2: Create root package.json
- name: "town-meeting-manager"
- private: true
- No workspaces field in package.json (pnpm uses pnpm-workspace.yaml instead)
- Add scripts:
  - "build": "turbo run build"
  - "dev": "turbo run dev"
  - "lint": "turbo run lint"
  - "test": "turbo run test"
  - "typecheck": "turbo run typecheck"
  - "clean": "turbo run clean && rm -rf node_modules"
  - "format": "prettier --write \"**/*.{ts,tsx,js,jsx,json,md}\""
  - "format:check": "prettier --check \"**/*.{ts,tsx,js,jsx,json,md}\""
- engines: { "node": ">=20.0.0", "pnpm": ">=9.0.0" }
- packageManager: "pnpm@9.15.4"

TASK 3: Create pnpm-workspace.yaml
```yaml
packages:
  - "packages/*"
```

TASK 4: Create packages/ directory structure
- packages/web/
- packages/mobile/
- packages/api/
- packages/shared/
- Create a placeholder .gitkeep in mobile/ since it won't be populated until Phase 2

TASK 5: Create per-package package.json files
Each package gets its own package.json:

packages/shared/package.json:
- name: "@town-meeting/shared"
- version: "0.0.1"
- private: true
- type: "module"
- main: "./dist/index.js"
- types: "./dist/index.d.ts"
- exports: { ".": { "types": "./dist/index.d.ts", "import": "./dist/index.js" } }
- scripts: build, clean, typecheck, lint, test
- Add "typescript" as a devDependency (use latest stable 5.x)

packages/web/package.json:
- name: "@town-meeting/web"
- version: "0.0.1"
- private: true
- type: "module"
- scripts: dev, build, clean, typecheck, lint, test
- Add "@town-meeting/shared" as a workspace dependency: "workspace:*"

packages/api/package.json:
- name: "@town-meeting/api"
- version: "0.0.1"
- private: true
- type: "module"
- scripts: dev, build, clean, typecheck, lint, test
- Add "@town-meeting/shared" as a workspace dependency: "workspace:*"

packages/mobile/package.json:
- name: "@town-meeting/mobile"
- version: "0.0.1"
- private: true
- scripts: placeholder only — "dev": "echo 'Mobile package is Phase 2'"
- Add "@town-meeting/shared" as a workspace dependency: "workspace:*"

TASK 6: Create tsconfig.base.json at the repo root
- compilerOptions:
  - target: "ES2022"
  - module: "ESNext"
  - moduleResolution: "bundler"
  - lib: ["ES2022"]
  - strict: true
  - esModuleInterop: true
  - skipLibCheck: true
  - forceConsistentCasingInFileNames: true
  - resolveJsonModule: true
  - isolatedModules: true
  - declaration: true
  - declarationMap: true
  - sourceMap: true
  - noUncheckedIndexedAccess: true
  - noUnusedLocals: true
  - noUnusedParameters: true
  - exactOptionalPropertyTypes: false (too aggressive for early development)
  - paths: { "@town-meeting/shared/*": ["./packages/shared/src/*"] }

TASK 7: Create per-package tsconfig.json
Each extends the base config:

packages/shared/tsconfig.json:
- extends: "../../tsconfig.base.json"
- compilerOptions: { outDir: "dist", rootDir: "src", composite: true }
- include: ["src/**/*"]
- exclude: ["dist", "node_modules", "**/*.test.ts"]

packages/web/tsconfig.json:
- extends: "../../tsconfig.base.json"
- compilerOptions: { jsx: "react-jsx", outDir: "dist", rootDir: "src", lib: ["ES2022", "DOM", "DOM.Iterable"] }
- include: ["src/**/*"]
- exclude: ["dist", "node_modules"]
- references: [{ path: "../shared" }]

packages/api/tsconfig.json:
- extends: "../../tsconfig.base.json"
- compilerOptions: { outDir: "dist", rootDir: "src" }
- include: ["src/**/*"]
- exclude: ["dist", "node_modules"]
- references: [{ path: "../shared" }]

packages/mobile/tsconfig.json:
- extends: "../../tsconfig.base.json"
- compilerOptions: { jsx: "react-jsx" }
- include: ["src/**/*"]
- exclude: ["dist", "node_modules"]
- references: [{ path: "../shared" }]

TASK 8: Install and configure Turborepo
- Install turbo as a root devDependency: pnpm add -D turbo
- Create turbo.json at the repo root:
```json
{
  "$schema": "https://turbo.build/schema.json",
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**"]
    },
    "dev": {
      "cache": false,
      "persistent": true
    },
    "lint": {
      "dependsOn": ["^build"]
    },
    "test": {
      "dependsOn": ["^build"]
    },
    "typecheck": {
      "dependsOn": ["^build"]
    },
    "clean": {
      "cache": false
    }
  }
}
```

- .idea/
- .vscode/ (but keep .vscode/settings.json if it exists)
- *.log

TASK 10: Create .nvmrc
Contents: "20" (Node.js 20 LTS)

TASK 11: Create .npmrc

```

TASK 12: Install dependencies and verify
- Run `pnpm install` at the repo root
- Verify pnpm-lock.yaml is created
- Create a minimal packages/shared/src/index.ts with a single export: `export const APP_NAME = "Town Meeting Manager";`
- Run `pnpm turbo build` and verify it completes successfully
- Run `pnpm turbo typecheck` and verify no TypeScript errors

VERIFICATION:
1. `ls packages/` shows web/, mobile/, api/, shared/
2. `pnpm ls -r` shows all four workspace packages
3. `pnpm turbo build` succeeds with no errors
4. `cat packages/shared/dist/index.js` contains the exported constant
5. `cat packages/shared/dist/index.d.ts` contains the type declaration
6. `node -e "console.log(require('./packages/shared/dist/index.js'))"` or ESM equivalent prints the constant
7. No TypeScript errors reported by `pnpm turbo typecheck`
```
