# Session 08.01 — Tests: Agenda Building & Live Meeting Manager

**Phase:** 1 — MVP | **Block:** 08 — Testing: Agenda & Meetings
**Dependencies:** 07.03, 05.01
**Estimated tasks:** 12

---

## Description

Write comprehensive tests for the agenda building system (Block 06) and the live meeting manager (Block 07). This includes component tests for template editing, agenda building, motion capture, vote recording, and executive session flows; integration tests for quorum enforcement and PDF generation; and end-to-end tests for the full meeting lifecycle and offline operation.

## Tasks

1. Write component tests for AgendaTemplateEditor: create template, add/remove sections, reorder, set default
2. Write component tests for AgendaBuilder: add agenda items, sub-items, drag-drop reorder, exhibit upload
3. Write integration test for agenda packet PDF generation: verify Puppeteer generates valid PDF with correct content
4. Write component tests for LiveMeetingPage: start meeting, attendance recording, agenda navigation
5. Write component tests for MotionCaptureDialog: create motion, validate required fields
6. Write component tests for VotePanel: record votes, calculate results, handle recusals
7. Write integration test for quorum enforcement: recusal drops below quorum -> alert -> block vote
8. Write component tests for executive session flow: enter/exit, legal citation required
9. Write component tests for adjournment: both methods (motion and chair declaration)
10. Write E2E test: full meeting lifecycle (create meeting -> build agenda -> start -> motions/votes -> adjourn -> post-review)
11. Write E2E test: offline meeting operation (disconnect -> record motion/vote -> reconnect -> verify sync)
12. Run full test suite and verify all tests pass
