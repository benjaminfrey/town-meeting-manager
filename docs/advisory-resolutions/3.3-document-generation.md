# Advisory 3.3 — Document Generation Strategy

**Advisory Item:** Document Generation Strategy
**Date:** March 2026
**Status:** Resolved — Puppeteer for complex documents, pdf-lib for simple ones

---

## Decision

**Puppeteer (HTML-to-PDF) for agenda packets and minutes** where layout quality matters. **pdf-lib for simpler documents** like meeting notices where clean text output is sufficient.

### Rationale

- **Agenda packets and minutes** are the public-facing, legally significant documents. They need professional layout, consistent formatting, and ADA-compliant structure (tagged PDFs). Puppeteer renders HTML/CSS in a headless browser, giving full control over typography, headers/footers, page breaks, and table layouts.
- **Meeting notices and simple documents** don't need complex layout. pdf-lib generates these quickly without spinning up a browser instance — lighter on resources and faster.

### Implementation Notes

- **Puppeteer memory management:** Run Puppeteer in a dedicated Docker container with memory limits. Launch a single browser instance and reuse it across document generation requests rather than spawning a new browser per document.
- **HTML templates:** Document templates are HTML/CSS files with Handlebars (or similar) variable substitution. This means document layout can be adjusted by editing CSS without changing application code.
- **Accessible PDFs:** Use semantic HTML (`<h1>`, `<table>`, `<thead>`, `<th>`, `<article>`, etc.) so Puppeteer produces tagged PDFs that meet WCAG 2.1 AA requirements.
- **pdf-lib usage:** For programmatic PDF generation of simple text documents — no browser overhead needed.

### Which Tool for Which Document

| Document | Tool | Rationale |
|----------|------|-----------|
| Agenda packet | Puppeteer | Complex layout, multiple sections, exhibits, page numbers |
| Meeting minutes | Puppeteer | Formatted narrative/action text, motion blocks, signature area |
| Meeting notice | pdf-lib | Simple text with date, time, location |
| Warrant articles | Puppeteer | Structured legal text with article numbering |
| Export/print of single agenda item | pdf-lib | Simple structured text |
